## 🎯 项目特点

1. **硬件级键盘模拟** - 使用 DirectInput 技术
2. **支持主流游戏启动器** - 战网、Steam、Riot Vanguard 等
3. **快捷键触发** - Ctrl+Alt+J 一键输入
4. **管理员权限** - 自动请求确保有效
5. **开箱即用** - 提供打包好的 exe 文件

推送代码时需要使用 Token 而不是密码：
1. 访问: https://github.com/settings/tokens
2. 生成新 token（勾选 repo 权限）
3. 复制 token 作为密码使用
